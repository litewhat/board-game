# BOARD-GAME

Console board game.

## Instalation

### 1. Docker
Make sure you have a docker-compose installed: https://docs.docker.com/compose/install/

### 2. Run the following commands:

```bash
    ~$  docker-compose up --build

```
### 3. Enter the container:

```
    ~$  docker-compose run game bash
    ~$  ./run
```
